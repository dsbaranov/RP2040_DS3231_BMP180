# Описание проекта
## Цель:
Создание автоматизированного устройства контроля подачи воды на огородном участке чтобы упростить жизнь моей бабушке.

## Опционально:
- Практика с CPP. В данном случае - на контроллере.

# Строение
## Физический уровень:
- Модуль реального времени (DS3231)
- Датчик температуры воздуха (BMP180)
- Датчик влажности (?)
- Датчик давления (BMP180)
- Модуль памяти (есть на примете)
- Дисплей (SSD1315)
- Кнопки (?)

## Уровень интерфейсов: 

RP2040 - I2C -> BMP180 -> SSD1315 -> DS3231 (WIP)
RP2040 - SPI -> <FLASH>
