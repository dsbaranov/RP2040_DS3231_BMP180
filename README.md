# Описание проекта
## Цель:
Создание автоматизированного устройства контроля подачи воды на огородном участке чтобы упростить жизнь моей бабушке.

## Дополнительные цели:
- Практика с CPP. В данном случае - на контроллере.
- Практика прошивки/отладки контроллера.
- Практика с OpenOCD, gdb-multiarch, arm-none-eabi-gcc, cmake, ninja и проч.


## Примерный функционал:
Устройство будучи развёрнутым на местности, будет самообучаться девиациям погоды и, вычисляя изменения(или их отсутствиие) атмосферных и временных параметров регулировать подачу воды к растениям.


# Строение
## Физический уровень:
- Модуль реального времени (DS3231)
- Датчик температуры воздуха (BMP180)(Неточный, надо искать аналог)
- Датчик влажности (?)
- Датчик давления (BMP180)
- Модуль памяти (W25QXX) W25Q32BVS
- Дисплей (SSD1315)
- Кнопки (?)
- Ключ управления цифровым гидрозатвором (?)
- Цифровой гидрозатвор нормально-закрытого типа. (?)

## Уровень интерфейсов: 

1. RP2040 - I2C -> BMP180, SSD1315, DS3231
2. RP2040 - SPI -> W25QXX

4. RP2040 -> UART -> CLI
